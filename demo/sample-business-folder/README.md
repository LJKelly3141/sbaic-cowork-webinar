# sample-business-folder

This folder is the sandbox Cowork points at during Demo 1.

**Populate it before the webinar** with 3–5 sanitized small-business files:

- A sample invoice (PDF or DOCX)
- An email saved as `.txt` or `.eml`
- A one-page memo (DOCX)
- A small spreadsheet (XLSX)
- Optional: a meeting agenda or note

All files should be **sanitized** — no real client info, no PII.

Delete this README before going live so Cowork doesn't index it.
